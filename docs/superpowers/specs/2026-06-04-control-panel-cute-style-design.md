# 控制面板浅色可爱风格规格

日期：2026-06-04

## 目标

把第二版控制面板从默认 PyQt 外观改成浅色可爱风格。功能不变，只优化布局、颜色、控件层级和中文显示。

## 视觉方向

- 背景使用浅粉白色。
- 每个功能区使用白色圆角卡片。
- 主按钮使用粉色，普通按钮使用浅蓝/浅灰，危险按钮使用红色。
- 滑块显示当前数值，滑槽和滑块颜色柔和。
- 底部状态提示使用浅色状态条。

## 布局

- 顶部增加标题和一句简短说明。
- 素材区、透明设置、大小、动作、对话分成独立卡片。
- 动作按钮保持两行三列。
- 对话输入框和按钮保持同一区域。
- 退出按钮放在底部。

## 非目标

- 不改变上传 MP4、透明处理、缩放、动作、聊天等功能逻辑。
- 不新增第三方 UI 库。
- 不改宠物窗口绘制逻辑。
