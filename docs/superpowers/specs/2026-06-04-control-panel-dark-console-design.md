# 控制面板深色产品控制台规格

日期：2026-06-04

## 目标

参考用户提供的深色产品面板截图，将控制面板从浅色卡片风改成深色双栏控制台。功能保持不变，只调整布局、视觉层级和中文显示。

## 视觉方向

- 深色背景和深灰卡片。
- 左侧为素材栏，显示当前素材和默认素材卡片。
- 右侧为预览和参数区，包含大小、透明容差、背景色、动作和对话。
- 主按钮使用橙色，状态标签使用绿色，普通按钮使用白色或深灰。
- 整体圆角、低饱和暗色、强调色清晰。

## 非目标

- 不做多个素材库管理。
- 不加入直播头像、点赞、字幕等参考图中的无关元素。
- 不改变 MP4 上传、透明处理、缩放、动作和对话逻辑。
